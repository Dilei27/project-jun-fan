# Product Language — Jun Fan

## Identidade

**Jun Fan** é a marca da interface. **Project Jun Fan** é a metodologia e fundação. **Horizon Design System** é a implementação visual.

## Tom de Voz

| Contexto | Tom |
|---|---|
| Interface | Português (pt-BR), claro e direto |
| Código | Inglês |
| Documentação | Português técnico |
| Erros | Informativos, não técnicos |

## Princípios Visuais

- Escuro como padrão
- Uma cor de acento por tela
- Tipografia Inter
- Motion funcional e discreto
- Componentes com borda sutil e cantos arredondados (8px-16px)
- Superfícies em camadas (base → elevada → soft)
