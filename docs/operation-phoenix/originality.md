## Elementos que Poderiam Ser Exclusivos da Marca

### 1. "The Strip" — Barra de Status do Ecossistema
Hoje o `StatusStrip` mostra 3 produtos com pontos. Poderia ser um elemento de assinatura visual: sempre presente no header ou topo de páginas principais, com os 3 pontos coloridos pulsando suavemente.

### 2. Terminal Typography
O Boot Loader usa terminal aesthetic. Poderíamos estender isso para:
- Blocos de código com "junfan" prompt
- Linhas de comando reais em vez de bullet points
- ">" como prefixo de seção

### 3. Glow por Produto
Cada produto já tem accent color. O hero do CC usa glows (azul + teal). Poderíamos ter um glow system: QA = azul, WhatsApp = teal, Vigilante = laranja.

### 4. Timeline como Navegação Principal
A timeline 2018-2025 é uma história. Se ela fosse o elemento central de navegação (não apenas uma seção), o produto seria inconfundível.

### 5. "Product Operating System" como Tagline
"Product Operating System" aparece no metadata mas não na interface. É um conceito forte e único.
