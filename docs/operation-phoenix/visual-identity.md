## Perguntas & Respostas

### O produto possui assinatura visual?
**Parcialmente.** O dark theme com azul #4F8CFF como acento principal, JF logo no header, e bordas sutis são um começo. Mas não há um elemento que seja **inequivocamente Jun Fan**.

### Se uma screenshot aparecer no LinkedIn, alguém reconheceria?
**Não.** Uma screenshot da home poderia ser confundida com:
- Um portfólio Next.js genérico
- Um Linear clone
- Um template de startup escuro
- Um dashboard de ferramenta interna

### O que falta para isso acontecer?
1. **Elemento visual de assinatura** — um pattern, gradiente, forma geométrica ou grid que seja "a marca registrada" do Jun Fan
2. **Tipografia mais distinta** — Inter é funcional, mas usada por 80% dos produtos modernos
3. **Logo mais presente** — "JF" no header é pequeno e discreto demais
4. **Cores de acento mais usadas** — hoje #4F8CFF aparece em links e badges, mas não como elemento de fundo ou destaque
5. **Motion característico** — nenhuma animação atual é memorável

---

## O Que É Único Hoje

- **Nome "Jun Fan"** — incomum, memorável
- **"QA Command Center"** como conceito de hub
- **Boot Loader** — tentativa de personalidade
- **Glows sutis** no hero do CC (bg-accent-qa/5 blur-3xl)
- **JF logo** com fundo azul no header

---

## O Que Parece Comum

- Layout de grid 3 colunas (qualquer template)
- Cards com border e bg (qualquer design system)
- Header fixo com nav links (qualquer site)
- Botões primário + outline (qualquer lib)
- Fonte Inter (qualquer produto moderno)
- Badges de status (qualquer dashboard)

---

## Propostas de Assinatura Visual (Sem Implementar)

1. **"Barra de Progresso do Ecossistema"** — um elemento visual horizontal que mostra o "health" dos 3 produtos, como uma linha pulsante com as cores dos produtos
2. **Grid Geométrico de Fundo** — um pattern sutil de grid hexagonal ou linhas conectadas no fundo do hero, que remeta a "engenharia" e "conexão entre módulos"
3. **Avatar Técnico** — uma representação visual do autor (foto + diagrama técnico) que apareça na home e no CC
4. **Cada Produto com um "Glow" Característico** — QA azul, WhatsApp teal, Vigilante laranja — os glows existem no hero do CC, mas poderiam ser mais presentes
