# Branding Agent — Validation Checklist

## Brand Guidelines
- [ ] Brand essence statement is written and current
- [ ] Voice is defined with 4-5 adjectives
- [ ] Tone is defined per context (marketing, product UI, errors, empty states)
- [ ] Visual identity rules are documented

## Color System
- [ ] Primary palette has at least 3 shades (light, base, dark)
- [ ] Neutral palette supports full grayscale (50-900)
- [ ] Semantic colors are defined (success, warning, error, info)
- [ ] All text/background pairs meet WCAG 2.1 AA contrast (4.5:1 normal, 3:1 large)
- [ ] Each color has a documented usage rationale
- [ ] No two colors serve the same purpose

## Typography
- [ ] Heading typeface is selected (max 2 families total)
- [ ] Body typeface is selected
- [ ] Type scale is defined with at least 6 sizes
- [ ] Line heights are specified for each size
- [ ] Font weights available are documented
- [ ] Monospace typeface is selected (if applicable)

## Logo & Icons
- [ ] Logo clear space is defined
- [ ] Minimum logo size is defined
- [ ] Prohibited logo treatments are listed (no stretching, no recoloring, no effects)
- [ ] Icon style is defined (outline vs filled, stroke width, corner radius)

## Brand Alignment Review
- [ ] Design feels minimal — no decorative elements without purpose
- [ ] Spacing is generous and consistent
- [ ] No skeuomorphism or heavy gradients
- [ ] Brand feels cohesive across all pages and touchpoints
- [ ] Third-party assets (illustrations, icons) match brand style
