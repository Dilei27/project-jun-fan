# Branding Agent — Usage Examples

## Example 1: Brand Voice Definition

**Brand Voice:** Confident. Clear. Human. Precise.

| Context | Tone | Example |
|---|---|---|
| Marketing | Bold, aspirational | "Absorb. Refine. Build." |
| Product UI | Helpful, direct | "Filter by technology" — not "You can choose to filter..." |
| Error states | Calm, actionable | "Something went wrong. Try refreshing." — not "Error 0x87ABC" |
| Empty states | Warm, encouraging | "No projects yet — check back soon." — not "No results found." |

## Example 2: Color Palette Philosophy

**Why these colors?**

- **Primary (Indigo):** Represents depth and trust. Inspired by traditional Japanese indigo dyeing (ai-zome). Not too warm, not too cool — balanced.
- **Neutral (Warm Gray):** Slightly warm gray to feel approachable and human. Pure gray (#808080) is cold and institutional.
- **Accent (Amber):** Used only for highlights and special states. Small doses create emphasis. Overuse would dilute brand minimalism.

**Rejected alternatives:**
- Pure blue (#2563EB) — too corporate, feels like a bank
- Pure gray (#6B7280) — too cold, feels like a utility
- Multiple accent colors — violates minimalism principle

## Example 3: Brand Review — Feature Page

**Feature reviewed:** New case studies page hero section

**Issues found:**
1. ❌ Hero image has a gradient overlay — decorative, not functional
2. ❌ Title uses two colors (primary + neutral) — typography alone should create hierarchy
3. ❌ Spacing between title and metadata is 16px — should be 24px (brand standard)
4. ✅ Body copy uses correct voice (confident, direct)
5. ✅ Logo placement respects clear space

**Resolution:**
- Remove gradient overlay
- Title uses single color (`color-neutral-900`) with font-weight and scale for hierarchy
- Increase spacing to 24px
- ✅ Approved after fixes
