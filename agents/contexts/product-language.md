# Product Language & Brand Identity

## Brand Identity

- **Name:** Jun Fan (Bruce Lee's given name)
- **Voice:** Clear, confident, direct. Speaks with the precision of a martial artist and the clarity of a craftsman.
- **Personality:** Disciplined but not rigid. Minimalist but not cold. Strong but not aggressive.

## Tone

| Context | Tone |
|---------|------|
| User-facing UI | Clear, concise, helpful |
| Error messages | Direct, actionable, no blame |
| Documentation | Thorough, precise, example-driven |
| Marketing / Landing | Bold, aspirational, energetic |
| Internal / Code comments | Technical, minimal, purposeful |

## Visual Principles

1. **Space is a material** — Generous whitespace creates hierarchy and calm.
2. **Typography leads** — Content hierarchy is established through type scale, not color or decoration.
3. **Color with purpose** — Every color has a semantic role. No decoration for its own sake.
4. **Motion that serves** — Animation clarifies relationships and provides feedback. It never distracts.
5. **Consistency is trust** — Users should feel the same patterns throughout every product.

## Writing Guidelines

- Prefer the active voice: "The system saves your work" not "Your work is saved by the system"
- Use "you" to address the user directly
- Keep sentences under 25 words where possible
- Use serial (Oxford) commas
- Capitalize proper nouns (Jun Fan, Horizon, Tailwind, etc.)
- Use lowercase for generic references (the app, the system, the component)
