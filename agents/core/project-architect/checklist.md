# Project Architect — Validation Checklist

## Vision & Roadmap
- [ ] Is the project vision statement current and unambiguous?
- [ ] Does the roadmap reflect the latest priorities?
- [ ] Are milestones decomposed into concrete, verifiable outcomes?

## Decisions
- [ ] Is every ADR up to date with the latest decision?
- [ ] Does each decision include rationale and impact?
- [ ] Are superseded ADRs clearly marked as deprecated?

## Cross-Agent Coordination
- [ ] Have all affected agents been notified of decisions?
- [ ] Are there any conflicting directives between agents?
- [ ] Is there a single source of truth for current priorities?

## Stakeholder Alignment
- [ ] Do current goals match stakeholder expectations?
- [ ] Are acceptance criteria testable and unambiguous?
